# Student Tools Hub

This project is a web application that helps students manage their tasks and improve their productivity.

## Installation

To get started with the project, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/AMBalaram/student-tools-hub.git
cd student-tools-hub
npm install
```

## Usage

To run the application in development mode, use:

```bash
npm run dev
```
